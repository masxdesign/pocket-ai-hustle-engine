# Pocket AI Hustle — Project Instructions

## Classroom Content: Language Structure

All classroom content must be organized by language inside the `/classroom/` directory.

**Structure:**
```
classroom/
  en/   ← English content
  es/   ← Latin American Spanish content
  [lang-code]/  ← any future languages
```

**Rules:**
- When creating or translating classroom content, always place it in the correct language subfolder.
- Translated content mirrors the same folder/file structure as the source language.
- Example: `classroom/en/AI Content Creator/module-1-lesson-1-ai-is-leverage.md` → `classroom/es/AI Content Creator/module-1-lesson-1-ai-is-leverage.md`
- Use Latin American Spanish (not Spain Spanish) for all `es/` content.
